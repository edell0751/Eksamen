Colaborating with GitHub

Colaborating with GitHub has for the most part been a straight forward process. We met some issues, mostly in the beginning, but when we got to know the features and its functionality we made it work. We had a few hiccups along the way, mostly merge related. For example one occation when our whole project got set back 2 weeks, because an old version of the website got merged into main. We had a backup and eventually solved the problem. Other than that we found GitHub to be a very powerfull tool for colaborating with other developers. It is really helpfull to se a full log of the merge history, with comments that summarize which changes have been made.

List of benefits we found using GitHub

  • Easy to cooporate
  • Branches made distributing work easier
  • GitHub pages made it possible to problem search our website with tools such as Wave
  • Intuitive and clear way of showing changes made
  • Given us insight on what it is like to work in the industry
  • Version control tool has been helpful on multiple occasions
